# Install

Use either of these repositories:

```
repositories {
    jcenter()
}
```

Or:

```
repositories {
    maven {
        url 'https://dl.bintray.com/novacrypto/Hashing/'
    }
}
```

Add dependency:

```
dependencies {
    compile 'io.github.novacrypto:Hash160:2018.10.06@jar'
}

```
